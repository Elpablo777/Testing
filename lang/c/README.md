# C

## Kurzbeschreibung
C ist eine der ältesten und einflussreichsten Programmiersprachen, ideal für Systemprogrammierung und Embedded-Entwicklung.

## Minimalbeispiel
```c
#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}
```

- `main.c`
- `Makefile`
